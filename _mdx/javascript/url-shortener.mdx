Have you ever wanted to share a long URL with someone, but didn't want to take up too much space in a text message or social media post? A URL shortener can help with that!

<h1>Create a URL Shortener</h1>
<h2>Requirements</h2>
<ul>
  <li>A form where users can input a long URL</li>
  <li>A way to generate a shortened URL for the inputted URL</li>
  <li>A way to store the mapping between the long and short URLs</li>
  <li>
    A way to redirect users to the original long URL when they visit the short
    URL
  </li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add a feature that allows users to customize the shortened URL (e.g.
    by choosing a specific keyword)?
  </li>
  <li>
    Can you add a feature that displays the number of times each short URL has
    been visited?
  </li>
</ul>
