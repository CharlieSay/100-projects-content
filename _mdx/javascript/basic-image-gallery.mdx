An image gallery is a common feature on websites that allows users to browse a collection of images. In this project, you will create a basic image gallery using HTML, CSS, and JavaScript.

<h1>Create a Basic Image Gallery</h1>
<h2>Requirements</h2>
<ul>
  <li>A grid of images that can be clicked to view a larger version</li>
  <li>A lightbox effect for displaying the larger version of the image</li>
  <li>Navigation buttons for moving between images in the gallery</li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add a feature that allows users to filter the images by tags or
    categories?
  </li>
  <li>
    Can you add a feature that allows users to upload their own images to the
    gallery?
  </li>
</ul>
