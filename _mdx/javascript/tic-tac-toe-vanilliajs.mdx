Tic Tac Toe is a classic game that has been enjoyed by people of all ages for generations. Now it's time to create your own version of this classic game using vanilla JavaScript!

<h1>Create a Tic Tac Toe game</h1>
<h2>Requirements</h2>
<ul>
  <li>A 3x3 grid where players can place their respective symbols (X or O)</li>
  <li>A way to alternate turns between players</li>
  <li>
    A way to determine when the game is over (e.g. when one player gets three in
    a row or the board is full)
  </li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add a feature that allows players to play against the computer?
  </li>
  <li>
    Can you add a feature that keeps track of the score and allows players to
    play multiple games?
  </li>
</ul>
