A calculator is a common tool that we use for basic arithmetic operations. In this project, you will create a simple calculator using vanilla JavaScript, HTML, and CSS.

<h1>Create a Simple Calculator</h1>
<h2>Requirements</h2>
<ul>
  <li>A display for showing the result of calculations</li>
  <li>
    Buttons for the digits 0-9 and the basic arithmetic operations (+, -, *, /)
  </li>
  <li>A way to clear the display and start a new calculation</li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>Can you add support for decimal numbers and the modulo operator?</li>
  <li>
    Can you add support for more advanced calculations, such as square roots and
    exponents?
  </li>
</ul>
