A text-based adventure game allows the user to navigate through a series of rooms and interact with objects and characters. In this project, you will create a simple text-based adventure game using Python.

<h1>Create a Simple Text-Based Adventure Game</h1>
<h2>Requirements</h2>
<ul>
  <li>A way for the user to navigate through a series of rooms</li>
  <li>
    A way for the user to interact with objects and characters in the game
  </li>
  <li>A way to keep track of the user's progress and inventory</li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add a feature that allows the user to choose their own path through
    the game?
  </li>
  <li>
    Can you add a feature that allows the user to solve puzzles or complete
    tasks to progress through the game?
  </li>
</ul>
<h2>Hint</h2>
<p>
  You can use functions and modules to organize and reuse code in this project.
