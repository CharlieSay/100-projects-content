Twitter is a popular social media platform where users can post short messages, called tweets, and interact with others. As a data scientist or developer, you may want to analyze the content and sentiment of tweets for a particular topic or user.

<h1>Create a Twitter Scraper</h1>
<h2>Requirements</h2>
<ul>
  <li>A way to scrape tweets from a specific user or hashtag</li>
  <li>
    A way to extract relevant information from the tweets (e.g. text, hashtags,
    mentions, etc.)
  </li>
  <li>
    A way to store the scraped tweets in a format suitable for further analysis
    (e.g. CSV, JSON, etc.)
  </li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add a feature that allows users to specify a date range for the
    tweets to be scraped?
  </li>
  <li>
    Can you add a feature that performs sentiment analysis on the scraped tweets
    and stores the results?
  </li>
</ul>
