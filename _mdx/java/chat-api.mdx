A chat application API allows users to send and receive messages in real-time using a server-side language and a websocket library like Socket.io. In this project, you will create a basic chat application API using Java and Socket.io.

<h1>Create a Basic Chat API</h1>
<h2>Requirements</h2>
<ul>
  <li>
    A way for users to send and receive messages in real-time using Socket.io
  </li>
  <li>A way to store the messages in a database</li>
  <li>
    A way to retrieve the messages from the database and display them to users
  </li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add authentication and authorization to the API, so that only
    authorized users can access the chat feature?
  </li>
  <li>
    Can you add support for group chats, where multiple users can participate in
    the same chat room?
  </li>
</ul>
<h2>Hint</h2>
<p>
  You can use a library like <a href="https://socket.io/">Socket.io</a> to
  implement the websocket functionality for this project.
</p>
