A stock market tracker application allows users to view and search for stock information using a web framework, a database, and a stock market API. In this project, you will build a simple stock market tracker application.

<h1>Create a Simple Stock Market Tracker</h1>
<h2>Requirements</h2>
<ul>
  <li>
    A way for users to view real-time stock information (e.g. price, volume,
    etc.)
  </li>
  <li>A way for users to search for specific stocks by ticker symbol</li>
  <li>A way to store stock information in a database</li>
  <li>A way to retrieve stock information from a stock market API</li>
</ul>
<h2>Bonus</h2>
<ul>
  <li>
    Can you add a feature that allows users to view historical stock data and
    trends?
  </li>
  <li>
    Can you add a feature that allows users to create and track portfolios of
    stocks?
  </li>
</ul>
<h2>Hint</h2>
<p>
  You can use a stock market API like the Yahoo Finance API or the Alpha Vantage
  API to retrieve real-time stock information for this project.
</p>
