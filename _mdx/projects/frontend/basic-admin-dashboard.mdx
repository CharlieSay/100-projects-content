In this project, we will be building a basic admin dashboard using a JavaScript framework such as React, Vue, or Angular. The dashboard will include features such as a user login system, a dashboard home page, and the ability to add, edit, and delete users.

<h2>Hint/Code snippet to start</h2>
To get started, you can use the following code snippet to set up a basic admin dashboard
using React:

```javascript
import React from 'react'
import { BrowserRouter as Router, Route } from 'react-router-dom'
import Login from './components/Login'
import Dashboard from './components/Dashboard'

function App() {
  return (
    <Router>
      <Route path="/login" component={Login} />
      <Route path="/dashboard" component={Dashboard} />
    </Router>
  )
}

export default App
```
